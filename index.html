Comment rejoindre le serveur ?

1 - Première étape

Tout d'abord, vous devez rejoindre le discord pour pouvoir lier votre compte Minecraft à votre compte Discord !

Discord

Ensuite, rejoignez le serveur Minecraft pour recevoir votre code qui vous permettra de lier vos comptes.

2 - Deuxième étape

Une fois vos comptes liés, vous devrez créer un compte sur le serveur en créant un mot de passe. Pour cela, vous devrez faire cette commande :

/register motdepasse motdepasse

Une fois votre compte créé, vous pourrez enfin jouer ! Lors de votre prochaine connexion, vous devrez vous connecter avec votre mot de passe en faisant :

/login motdepasse

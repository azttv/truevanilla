Comment rejoindre True Vanilla ?

Guide rapide pour te connecter au serveur

Suis ces étapes rapides pour rejoindre le serveur True Vanilla.

Étape 1 — Minecraft Java

Assure-toi d'avoir Minecraft Java Edition installé et à jour.

Étape 2 — Ajouter le serveur

Ouvre Multijoueur → Ajouter un serveur et renseigne l'adresse IP fournie par les admins.

Étape 3 — Connecte-toi

Sélectionne le serveur dans la liste et clique sur Rejoindre.
